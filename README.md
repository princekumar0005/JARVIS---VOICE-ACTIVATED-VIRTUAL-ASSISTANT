# 🤖 JARVIS - Voice Activated Virtual Assistant

JARVIS is a **Python-based voice-controlled virtual assistant** that can perform tasks like opening websites, playing music, fetching news, and answering questions using AI.

Inspired by real-world assistants like Alexa and Google Assistant.

---

## 🚀 Features

- 🎙️ Voice Recognition (SpeechRecognition)
- 🔑 Wake Word Detection ("Jarvis")
- 🔊 Text-to-Speech (gTTS + pyttsx3)
- 🌐 Open Websites (Google, YouTube, LinkedIn, Facebook)
- 🎵 Music Playback (Custom Music Library)
- 📰 Latest News Fetching (NewsAPI)
- 🤖 AI Responses using OpenAI API
- ⚡ Real-time Command Processing

---

## 🛠️ Tech Stack

- Python
- speech_recognition
- pyttsx3
- gTTS
- pygame
- requests
- OpenAI API
